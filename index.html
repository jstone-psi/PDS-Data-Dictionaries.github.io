The NASA Planetary Data System PDS4 Data Dictionaries Repo is used for managing PDS4 Local Data Dictionaries (LDDs) for the PDS4 Information Model. LDDs are a critical component of the PDS4 Data Standard.

This website is intended to provide a how-to for to developing, managing, and using PDS Local Data Dictionaries.
